# Contributing to BearClawiOS

## Setup

Open the Xcode project and use the app and test targets for development.

## Expectations

- Keep remote access secure by default.
- Preserve typed, user-visible failure modes for auth and TLS issues.
- Update `README.md` for product-facing behavior changes.
- Update `SECURITY.md` for token, pairing, or certificate changes.
- Update `CHANGELOG.md` for notable repo changes.

Active unfinished work belongs in the workspace root `ROADMAP.md`.
